# Skill: feature-implementation

## Goal

Deliver the requested feature with **minimal surface area** and **clear validation**.

## Method

1. Restate acceptance criteria in your own words.
2. Plan the smallest set of files to touch.
3. Implement with tests or a manual verification checklist.
4. Avoid drive-by refactors.

## Guardrails

- No unrelated formatting sweeps.
- No dependency additions unless required — if needed, justify in the plan.
